# Statistics for Machine Learning

---

## What is Statistics?

Statistics is the study of data — how to collect, analyze, interpret, and present it.

---

## What is Data?

Data is a collection of information.

It can be:
- Numbers  
- Images  
- Audio  
- Text  

---

## Why Do We Use Statistics in Machine Learning?

Statistics helps us:
- Understand data before applying algorithms  
- Identify patterns and relationships  
- Improve model performance  
- Make better decisions  

---

## Types of Data

### 1. Qualitative Data (Categorical)

Describes qualities or characteristics.  
Non-numerical (cannot perform mathematical operations)

Examples:
- Gender  
- Color  
- Blood group  

#### Types:

Nominal Data:
- Categories without order  
- Examples: Gender, Blood group, Colors  

Ordinal Data:
- Categories with order but no fixed scale  
- Examples: Education level, Ratings (poor → excellent)  

---

### 2. Quantitative Data (Numerical)

Consists of numbers.  
Mathematical operations can be performed  

Examples:
- Age  
- Height  
- Weight  
- Salary  

#### Types:

Discrete Data:
- Countable values (whole numbers)  
- Examples: Number of students, cars  

Continuous Data:
- Measurable values (can include decimals)  
- Examples: Height, weight, temperature  

---

## Extended Classification (Scales of Measurement)

Interval Data:
- Equal intervals between values  
- No true zero  

Examples:
- Temperature (°C, °F)  
- Calendar years  

Ratios are not meaningful  

---

Ratio Data:
- Equal intervals and a true zero  

Examples:
- Age  
- Weight  
- Income  

Ratios are meaningful

## Types of Statistics

### 1. Descriptive Statistics (Population)

Used to describe and summarize data  
No prediction, no inference  

Includes:

Measures of Central Tendency:
- Mean  
- Median  
- Mode  

Measures of Variability:
- Range  
- Variance  
- Standard Deviation  
- Interquartile Range (IQR)  

Frequency Distribution:
- Counts  
- Percentages  
- Histograms  

---

### 2. Inferential Statistics (Sample)

Used to make conclusions about a larger population using sample data  

Types:

Hypothesis Testing:
- Tests assumptions  
- Examples: t-test, z-test, chi-square test, ANOVA  

Estimation:
- Predict population values  
- Example: Confidence intervals  

Regression Analysis:
- Studies relationships between variables  
- Example: Predicting exam performance using study hours  

Correlation Analysis:
- Measures strength and direction of relationships  
- Example: Pearson correlation  

---

## Conclusion

Statistics is the foundation of Machine Learning. It helps in understanding data, preparing it for models, and making accurate predictions.
