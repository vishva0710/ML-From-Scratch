# What is Machine Learning?

Machine Learning is a branch of Artificial Intelligence that enables computers to learn patterns from data and make predictions or decisions without being explicitly programmed.

Machine Learning is widely used in:
- Recommendation Systems
- Spam Detection
- Image Recognition
- Chatbots
- Fraud Detection
- Medical Diagnosis
- Self-driving Cars

# Model Problems

## Underfitting
Underfitting happens when the model learns too little from the data.

Characteristics:
- Low accuracy
- Poor predictions
- High bias

---

## Overfitting
Overfitting happens when the model memorizes the training data instead of learning patterns.

Characteristics:
- Very high training accuracy
- Poor test accuracy
- High variance
