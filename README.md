# ML-From-Scratch

This repository documents my journey of revising and mastering Machine Learning concepts from scratch.

The goal of this project is to build a strong foundation by understanding the core concepts behind ML algorithms instead of just using libraries.

##   What this repo contains

-  Basic Statistics (Mean, Variance, Distribution, Correlation)
-  Feature Engineering (Scaling, Encoding, Handling Missing Values)
-  Feature Selection Techniques
-  Machine Learning Algorithms implemented from scratch:
  - Linear Regression
  - Logistic Regression
  - KNN
  - Naive Bayes
  - Decision Trees
  - Random Forest
  - SVM
  - Ensemble Learning

##  Objective

- Strengthen ML fundamentals
- Implement algorithms using Python (NumPy)
- Visualize concepts for better understanding
- Build mini-projects using learned algorithms
- Share learning consistently

##  Structure

Each topic/day includes:
- Theory explanation
- Implementation from scratch
- Visualization
- Clean code version

##  Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- sklearn

##  Progress

This is a continuous learning repository where new topics will be added regularly.
